# 消除特效图片目录

请把消除特效图片放入此目录。

## 建议的特效文件

| 文件名 | 用途 | 说明 |
|--------|------|------|
| `effect_match.png` | 普通消除特效 | 3连消除时播放 |
| `effect_match4.png` | 4连消除特效 | 4连消除时播放 |
| `effect_match5.png` | 5连消除特效 | 5连及以上消除时播放 |
| `effect_bomb.png` | 炸弹爆炸特效 | 炸弹触发时播放 |
| `effect_rocket.png` | 火箭发射特效 | 火箭触发时播放 |
| `effect_rainbow.png` | 彩虹球特效 | 彩虹球触发时播放 |
| `effect_combo.png` | 连击特效 | 连锁消除时播放 |
| `effect_select.png` | 选中特效 | 宝石选中时播放 |

## 格式要求
- 格式：PNG（支持透明背景）
- 建议尺寸：128x128 或 256x256 像素
- 动画：如果是序列帧，请按 `effect_xxx_01.png`, `effect_xxx_02.png` 命名

## 占位文件
以下文件为占位符，请替换为实际特效图片：
- effect_match.png
- effect_bomb.png
- effect_rocket.png
